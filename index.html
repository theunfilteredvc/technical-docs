Introduction

Python is a high-level, interpreted programming language known for its clear syntax and readability.

It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

print("Hello, World!")

Python is widely used in web development, data science, artificial intelligence, and more.

  • Simple syntax
  • Massive community
  • Cross-platform
Installation

To install Python, visit the official website and download the installer for your OS.

python --version

After installation, confirm the version in your terminal or command prompt.

Python comes with pip, the package installer, to manage libraries.

pip install package_name
  • Windows: Use MSI installer
  • macOS: Use Homebrew or installer
  • Linux: Use package manager (apt, yum)
Syntax and Indentation

Python uses indentation to define code blocks instead of braces.

This enforces clean and consistent code.

if x > 0:\n print("Positive")

Incorrect indentation results in a syntax error.

  • Use 4 spaces (PEP8)
  • No tabs mixed with spaces
  • Consistent style helps readability

Control Flow

Python supports conditional statements and loops.

Use if-elif-else for branching logic.

if x == 1:\n print("One")

Loops help iterate over data structures.

for i in range(5):\n print(i)
  • if / elif / else
  • for loops
  • while loops
  • break and continue
